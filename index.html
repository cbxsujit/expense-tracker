Google Sheets Integration

üìå How it works: Connect this app to Google Sheets to sync your expenses across all devices. Your data will be automatically saved to a Google Sheet that you can access anywhere.

Step 1: Create Your Google Sheet

1. Go to Google Sheets and open your existing spreadsheet (or create a new one)

2. Create a new tab called "Expenses" (the script will auto-create it if it doesn't exist)

3. In the first row of the Expenses tab, add these headers: Date, Amount, Category, Description, Notes, Timestamp

Note: The script will automatically create the "Expenses" tab and add headers if they don't exist.

Step 2: Set Up Google Apps Script

1. In your Google Sheet, go to Extensions ‚Üí Apps Script

2. Delete any code in the editor and paste the script provided below

3. Click Deploy ‚Üí New Deployment

4. Select type: Web app

5. Set "Who has access" to Anyone

6. Click Deploy and copy the Web App URL
